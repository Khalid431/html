JavaScript
A programming language used to add interactivity to websites.
DOM
Document Object Model — a programming interface for HTML documents.
Responsive Design
A design approach that makes websites work on all screen sizes.
Semantic HTML
HTML elements that clearly describe their meaning and purpose.
API
Application Programming Interface used for communication between software systems.
Frontend Development
The part of web development focused on user interfaces and browser interactions.
Backend Development
Server-side development that handles databases, authentication, and logic.
Framework
A pre-built structure that helps developers build applications faster.
Version Control
A system that tracks changes in code over time.
Git
A popular version control system used by developers.
HTTP
HyperText Transfer Protocol used for communication on the web.
HTTPS
A secure version of HTTP that encrypts data transmission.
Cookie
Small data stored in the browser to remember user information.
Local Storage
A browser feature that stores data permanently on the client side.

Practices that make websites usable for people with disabilities.
SEO
Search Engine Optimization — improving website visibility in search engines.

A CSS layout model for arranging items in rows or columns.
